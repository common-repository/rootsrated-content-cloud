== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rootsrated-content-cloud` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Matcha WP Plugin -> Settings page to enter API credentials
4. If the credentials are valid, you will get a notification stating your plugin is active
5. Begin scheduling content on Matcha!

== Frequently Asked Questions ==

= How do I find new content for my Wordpress site? =

Login to Matcha, and publish content to your Wordpress site. Once it's published, it should appear in Wordpress.

= What happens if Matcha has a new version? =

We will notify your Wordpress site that a new version exists, and you'll have the opportunity to accept or decline those changes.

= My credentials failed -- what should I do? =

Make sure your credentials are correct and entered as they appear on Matcha. If they are entered correctly, then contact support@getmatcha.com for help!
